# Sultana Shop Store

## Lancer le backend

1. Ouvrir un terminal dans `backend`
2. Installer les dependances:
   - `npm install`
3. Lancer l'API:
   - `npm start`

API disponible sur `http://localhost:4000/api`.

## Ouvrir le frontend

- Ouvrir `index.html` dans le navigateur (ou via Live Server).
